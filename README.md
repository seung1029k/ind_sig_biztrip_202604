# ind_sig_biztrip_202604

Indonesia / Singapore business trip and management portal pages.

## Live site

- GitHub Pages URL: `https://seung1029k.github.io/ind_sig_biztrip_202604/`
- If the URL does not open yet, enable `Settings > Pages > Source: GitHub Actions` in the repository once.

## Included pages

- `index.html`: GitHub Pages entry point
- `IDN_SG_Portal.html`: main portal
- `IDN_SG_HR.html`: HR overview
- `IDN_SG_Finance.html`: finance overview
- `trip_itinerary_apr2026.html`: business trip itinerary

## How to use

- Open the GitHub Pages URL on iPhone or iPad for the portal home.
- Use the `출장 일정표` tab inside the portal to open the trip itinerary overlay.
- You can also open `trip_itinerary_apr2026.html` directly if you want the schedule page only.

## Notes

- Source PDFs and extracted profile photos are intentionally excluded from this repository.
- The portal and HR pages will fall back to initials where local profile photos are not present.
- GitHub Pages deployment is configured with `.github/workflows/pages.yml`.
